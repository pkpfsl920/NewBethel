At STEM Camp. I've learned about microcontrollers. Micro controllers are basically a computer system on a chip that helps things to interact with each using a smart device and a cord. Some of the things that we've done with them are get a light to blink faster or slower, make a (very annoying) noise with a button, and we used an ultrasonic distance sensor to tell the distance between itself and a moving object in it's "view". We've used things like the breadboard, LED light, ultrasonic distance sensor, buttons, resistors, buzzors, and the arduino board.
